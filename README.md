# 🛒 장보기 리스트 앱

한국어 쇼핑 리스트 웹 애플리케이션입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 체크박스로 완료 표시 / 해제
- 아이템 개별 삭제
- 완료 항목 일괄 삭제
- localStorage로 데이터 영구 저장
- 총 개수 / 남은 개수 통계 표시

## 사용 방법

`shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 테스트 실행

```bash
npm install
node test-shopping.js
```

Playwright를 사용한 자동화 테스트 (총 9개 시나리오):
1. 초기 상태 확인
2. 아이템 추가
3. Enter 키로 추가
4. 빈 입력 방지
5. 체크(완료) 기능
6. 아이템 삭제
7. 완료 항목 일괄 삭제
8. localStorage 데이터 유지
9. 전체 삭제 후 빈 상태 복원
